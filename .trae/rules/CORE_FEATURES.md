# 核心功能

## 1. 智能对话提取

从小说文本中自动识别和提取对话内容，利用AI模型的理解能力，准确识别对话的开始和结束，以及说话者信息。

## 2. 多平台支持

集成多个AI平台，用户可以根据需要选择不同的模型：

| 平台 | 优势 | 配置参数 |
|------|------|----------|
| DeepSeek | 推荐，性能稳定 | DEEPSEEK_API, LLM_PLATFORM="deepseek" |
| OpenAI | 强大的模型能力 | OPENAI_API_KEY, LLM_PLATFORM="openai" |
| SiliconFlow | 提供多种开源模型 | SILICONFLOW_API_KEY, LLM_PLATFORM="siliconflow" |
| Kimi | 擅长长文本处理 | MOONSHOT_API_KEY, LLM_PLATFORM="moonshot" |

## 3. 并发处理

- 默认使用8个线程并发处理
- 支持自定义线程数量（-t参数）
- 线程安全的去重和结果写入
- 智能任务调度和负载均衡

## 4. 智能分块

- 基于token的文本分块
- 块间重叠保持上下文
- 长行自动分割
- 空行和多余空格清理

## 5. 自动去重

识别和移除重复对话，提高数据质量，避免重复数据对后续处理的影响。

## 6. 统计分析

生成详细的对话统计信息，包括：
- 总对话数和角色数量
- 角色对话分布
- 平均对话长度
- 文本块处理统计
- 错误处理汇总

## 7. 灵活输出

支持多种输出格式和选项：
- 包含chunk-id的标准格式
- 不含chunk-id的旧格式（向后兼容）
- 保存原始chunk文本
- 按chunk-id排序
- 生成旧格式文件
